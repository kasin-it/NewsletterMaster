## Local Setup

### Setup Instructions

1. Checkout to the "local" branch:

   ```bash
   git checkout local
   ```

2. Install project dependencies:

   ```bash
   pnpm i
   ```

3. Start Supabase:

   ```bash
   supabase start
   ```

4. Run the project in preview mode:

   ```bash
   pnpm run preview
   ```
