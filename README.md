## Local Setup

### Setup Instructions

1. Install project dependencies:

   ```bash
   pnpm i
   ```

2. Start Supabase:

   ```bash
   supabase start
   ```

3. Run the project in preview mode:

   ```bash
   pnpm run preview
   ```
