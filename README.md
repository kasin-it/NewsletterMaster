## Setup Instructions

    1. Clone this repository:

    ```bash
        git clone https://github.com/yourusername/sass-newsletter.git
    ```

    2. Install project dependencies:

    ```bash
        pnpm install
    ```


    3. Start the app:

    ```bash
        pnpm dev
    ```


    Interact with the API using tools like Altair or Insomnia.
